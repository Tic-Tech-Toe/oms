Badge-style wave — compact, app-icon friendly.

How to tweak

Open the downloaded SVG in Figma / Illustrator / VS Code. The gradient stops are #0078FF → #00D2A8. To change the palette, edit the stop-color values inside each <linearGradient>.

<linearGradient id="g1" x1="0" x2="1" y1="0" y2="1">
  <stop offset="0%" stop-color="#0078FF"/>
  <stop offset="100%" stop-color="#00D2A8"/>
</linearGradient>
